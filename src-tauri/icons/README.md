# Ícones do Aplicativo

Para gerar ícones personalizados para o AutoOS:

1. Crie uma imagem PNG de 1024x1024px com o logo do sistema
2. Use o gerador de ícones do Tauri:

```bash
npm run tauri icon caminho/para/sua/imagem.png
```

Isso irá gerar automaticamente todos os ícones necessários para Windows, macOS e Linux.

## Temporário

Por enquanto, os ícones padrão do Tauri serão usados. Atualize conforme necessário.
